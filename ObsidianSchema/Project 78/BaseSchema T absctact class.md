Абстрактный базовый класс для всех схем. Хранит набор проверок (checks) и управляет флагом required. 
Реализует общий алгоритм isValid, который последовательно применяет все добавленные проверки к переданному значению.

### Поля:
1) Map<String, Predicate<Т>> checks — коллекция в абстрактном классе, которое отвечает за проверки. 
2) boolean required — поле, отвечающее за необходимость присутствия данных.

### Методы:
1) boolean isValid(T value) — окончательный метод проверки значения по схеме.
2) void addCheck(String name, Predicate<Т>) — добавление проверки в коллекцию.
