Валидатор схема Integer значений

### Методы:
1) isValid(Integer intger) - проверка значения, наследуется от BaseSchema.
2) required() - проверяет дано ли значение вообще (Если был передан null и required не был задан - null пройдёт).
3) positive() - проверяет, положительное ли значение.
4) range(int minValue, int maxValue) - задаёт границы значений, границы входят в валидацию.
### Способ работы:
1) Вызывается валидатор.
2) Вызывается схема валидации (number()).
3) Задаются параметры
