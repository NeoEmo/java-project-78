Схема для валидации чисел (целых). Добавляет проверки: обязательность (required), 
положительность (positive) и вхождение в диапазон (range).

### Методы:
1) NumberSchema required() — требует наличия данных.
2) NumberSchema positive() — требует чтобы переданное число было положительным.
3) NumberSchema range(int min, int max) — требует, чтобы переданное число находилось в указанных значениях.
