Я оказался немного неправ и Validator будет что-то вроде фабрики для трёх схем, которые наследуются от базового интерфейса схем. Пустой конструктор public.
### Методы:
 1) string() - вызывает билдер схемы строки.
 2) number() - вызывает билдер схемы цифр.
 3) map() - вызывает билдер схем мап.
