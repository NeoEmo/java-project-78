Схема для валидации строк. Добавляет проверки: обязательность (required), 
минимальная длина (minLength) и наличие подстроки (contains).

### Методы:
1) StringSchema required() — требует наличия данных.
2) StringSchema minLength(int length) — требует чтобы переданная строка была больше по длине, чем переданное значение
3) StringSchema contains(String str) — требует, чтобы в переданное строке содержалось определённое значение\строка
