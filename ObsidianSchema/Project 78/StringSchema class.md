// Новая редакция

Своего рода билдер-валидатор схем строк.
### Методы:
1) IsValid(T) - в дальнейшем будет наследоваться от BaseSchema.
2) requared() - требование присутствия строки.
3) minLength(int length) - требование по длинне строки, не должно быть меньше переданного числа.
4) contains(String str) - требование по содержанию строки в isValid().
### Способ работы:
 1) Вызывается Validator.
 2) вызывается схема валидации.
 3) прописываются нужные параметры.
